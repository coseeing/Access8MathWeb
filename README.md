# Access8mathweb

## Sites
* Production: https://accessibility.twvip.org/Access8Math/
* Staging: https://access8math-stg.coseeing.org

## Prerequisites

- Node.js >= 18
- NPM >= 9

## Project setup

```
npm install
```

### Development

```
npm runs start
```

### Build for Production

```
npm run build
```

### Lint

```
npm run lint
```

### Test

```
npm run test
```
